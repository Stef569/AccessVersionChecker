This tool allows to get the specific access database version.

It can also check for comptability (Access files below version 97 can no longer be opened in access 2016)

It requires The DAO reference which is part of Microsoft Office. Office must be installed before running this tool.
also install Microsoft Access Database Engine 2010 Redistributable from https://www.microsoft.com/en-us/download/confirmation.aspx?id=13255
